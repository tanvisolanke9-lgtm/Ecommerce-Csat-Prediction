# CSAT Prediction using ANN

##  Overview

This project predicts Customer Satisfaction (CSAT) scores using an Artificial Neural Network (ANN) based on customer support data.

##  Workflow

* Data Cleaning & Handling Missing Values
* Feature Engineering & Encoding
* Model Building using ANN (Keras)
* Evaluation using MAE (~0.9–1.1)
* Insight Generation
* Local Deployment (Save, Load, Predict)

##  Results

The model captures overall trends in customer satisfaction but shows some limitations in predicting extreme values.

##  Technologies

Python, Pandas, NumPy, Scikit-learn, TensorFlow/Keras

##  Conclusion

The model successfully predicts CSAT scores and can be used for improving customer service through data-driven insights.
